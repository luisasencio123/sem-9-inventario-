# Sistema de Gestión de Inventarios

Proyecto desarrollado en Python aplicando Programación Orientada a Objetos.

Estructura:
- modelos/: Clase Producto
- servicios/: Clase Inventario
- main.py: Menú interactivo en consola

Permite:
- Añadir productos
- Eliminar productos
- Actualizar productos
- Buscar productos
- Listar inventario
